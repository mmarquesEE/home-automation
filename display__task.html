void display_task ( void *  ignore)

display_task

Função responsável pela exibição de informações no display OLED. Ela utiliza a biblioteca ´OLEDDisplay.h´ para controlar o display e as filas 'sensor_queue' e 'time_queue' Ela obtém os dados do sensor e o tempo atual das filas correspondentes e os exibe no display periodicamente a cada 100ms
