lamp_task

A tarefa lamp_task é responsável pelo controle das lâmpadas com base nos dados recebidos pela fila lamp_queue. Ela configura a direção dos pinos GPIO correspondente às lampadas e inicializa o estado com todas apagadas. No laço infinito ela obtem os dados mais recentes da fila ´lamp_queue´ por meio de iterações que vão atualizando estado. Se o estado for 'o': liga a lâmpada Caso contrário: desliga a lâmpada

Essa tarefa é realizada periodicamente a cada 100ms para verificar atualizações
